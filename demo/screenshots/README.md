# App Screenshots

These screenshots represent the UI of the **Local Workers App** — a React Native mobile app that connects customers with nearby skilled workers.

All screenshots are rendered at **390×844px** (iPhone 14 viewport) and reflect the app's earthy design language (teal `#2f6c62`, orange `#ca6b2c`, beige `#f4f1ea`).
